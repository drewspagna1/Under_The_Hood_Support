# Under The Hood — Support Site

This repo hosts the public support page for the **Under The Hood** iOS app via GitHub Pages.

**Live URL:** https://drewspagna1.github.io/Under_The_Hood_Support/

**Support URL (App Store):** https://drewspagna1.github.io/Under_The_Hood_Support/support.html

## Setup

1. Push the contents of this folder to the `main` branch of `drewspagna1/Under_The_Hood_Support`
3. Go to **Settings → Pages → Source** and set branch to `main`, folder to `/ (root)`
4. GitHub Pages will be live at the URL above within a few minutes

## Contents

- `index.html` — landing page / marketing page
- `support.html` — support & FAQ page (linked from App Store)
